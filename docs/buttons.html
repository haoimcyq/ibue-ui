按钮 BUTTON

普通按钮

为button、a等标签或元素添加按钮类 .button 即可使按钮样式生效。

示例代码：
<button class="button is-primary" role="button">primary</button>
...
演示效果：
颜色类 .is-primary .is-secondary .is-success .is-error .is-warning .is-info .is-dark .is-light
含义 主要 次要 成功 失败 提醒 信息 暗色 亮色
效果

朴素按钮

给按钮类 .button 添加 .is-plain 类生成朴素按钮样式。

示例代码：
<button class="button is-primary is-plain" role="button">primary</button>
...
演示效果：
颜色类 .is-primary .is-secondary .is-success .is-error .is-warning .is-info .is-dark .is-light
含义 主要 次要 成功 失败 提醒 信息 暗色 亮色
效果

禁用状态

按钮尺寸

给按钮类 .button 设置尺寸。

示例代码：
<span class="button is-primary is-mini" role="button">primary</span>
...
演示效果：
尺寸类 字体大小 高度 效果
is-mini 12px 24px button
is-small 14px 32px button
默认 14px 36px button
is-medium 16px 40px button
is-large 18px 48px button

块级按钮

给按钮类 .button 添加 .is-fluid 类设置块级按钮。

示例代码：
<span class="button is-primary is-fluid" role="button">primary</span>
...
演示效果：
primary
success

图标按钮

给按钮类 .button 添加 .is-icon 类设置为纯图标型按钮。如果是文字图标组合则直接给按钮类 .button 插入图标元素或标签即可。

示例代码：
<span class="button is-primary is-icon" role="button"><i class="iconfont icon-icon-test7"></i></span>
<span class="button is-primary" role="button"><i class="iconfont icon-icon-test7"></i> primary</span>
...
演示效果：
primary

加载中

给按钮类 .button 添加 .is-loading 类设置加载状态。

示例代码：
<span class="button is-primary is-plain is-loading" role="button"></span>
<span class="button is-primary is-loading" role="button"></span>
...

按钮组

给按钮类 .button 的父元素或标签添加 .button-group 类设置为按钮组。

示例代码：
<div class="button-group">
    <button class="button is-plain is-primary is-active" role="button">primary</button>
    <button class="button is-plain is-primary" role="button">primary</button>
    ...
</div>
