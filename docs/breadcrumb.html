路径导航 BREADCRUMB

基本路径导航

路径导航类默认分割线为斜杆。

示例代码：
<ul class="breadcrumb">
    <li><a href="#">首页</a></li>
    <li><a href="#">组件</a></li>
    <li class="active">路径导航</li>
</ul>
演示效果：
  • 首页
  • 组件
  • 路径导航

分割线

给路径导航类 .breadcrumb 添加 .is-arrow 类设置分割线为箭头。

示例代码：
<ul class="breadcrumb is-arrow">
    <li><a href="#">首页</a></li>
    <li><a href="#">组件</a></li>
    <li class="active">路径导航</li>
</ul>
演示效果：
  • 首页
  • 组件
  • 路径导航
