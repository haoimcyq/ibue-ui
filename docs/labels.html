标签 LABLE

普通标签

添加标签类 .label 即可使标签样式生效。

示例代码：
<span class="label is-primary">label</span>
...
演示效果：
颜色类 .is-primary .is-secondary .is-success .is-error .is-warning .is-info .is-dark .is-light
含义 主要 次要 成功 失败 提醒 信息 暗色 亮色
效果 label label label label label label label label

朴素标签

给标签类 .label 添加 .is-plain 设置朴素标签样式。

示例代码：
<span class="label is-primary is-plain">label</span>
...
演示效果：
颜色类 .is-primary .is-secondary .is-success .is-error .is-warning .is-info .is-dark .is-light
含义 主要 次要 成功 失败 提醒 信息 暗色 亮色
效果 label label label label label label label label
